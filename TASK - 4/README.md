SPAM SMS DETECTION 
1.Overview :
- This project classifies SMS messages as Spam or Not Spam using Natural Language Processing (NLP) and Machine Learning.

2.Technologies used:

- Python
- Pandas
- NLTK
- Scikit-learn
- TF-IDF Vectorization

3.Workflow :
- Data Cleaning
- Text Preprocessing
- TF-IDF Vectorization
- Model Training-
- Spam Prediction
- Model Evaluation

4.Results :
- Accuracy: 95%
- Sample Predection Input: "Congratulations! You have won a free iPhone." Output: Spam
